# API Reference

The complete API reference is generated from TypeScript sources using TypeDoc.

The complete API reference is generated from TypeScript sources using TypeDoc.

## [Viewing the API Reference](https://rezitui.dev/docs/api\#viewing-the-api-reference)

### [Online](https://rezitui.dev/docs/api\#online)

The hosted API reference is available at the documentation site under `/api/reference/`.

- [Open API reference](https://rezitui.dev/docs/api/reference/index.html)

### [Local Build](https://rezitui.dev/docs/api\#local-build)

Generate the API documentation locally:

```
npm run docs:api
```

The output is written to `out/typedoc/index.html`.

Alternatively, build the complete documentation site:

```
npm run docs:build
```

The API reference is included at `out/site/api/reference/index.html`.

## [Quick Reference](https://rezitui.dev/docs/api\#quick-reference)

For the most common APIs, see:

- [Widget Catalog](https://rezitui.dev/docs/widgets/index) \- All widget types and props

- [Hooks Reference](https://rezitui.dev/docs/guide/hooks-reference) \- `ctx.useTheme()` and composition hooks
- [@rezi-ui/core](https://rezitui.dev/docs/packages/core) \- Core package exports
- [@rezi-ui/node](https://rezitui.dev/docs/packages/node) \- Node.js/Bun backend
